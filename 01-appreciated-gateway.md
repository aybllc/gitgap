# The Appreciated Gateway

**Definition:** A result passes the Appreciated Gateway when the qualitative idea is already broadly held but no explicit quantitative specification yet exists. The paper's contribution is the first number — not the concept.

This is distinct from:
- **Origination** — the paper created the structure
- **Discovery** — the paper identified a previously unmeasured relation
- **Centerpiece** — the paper cannot survive without this result

A result can pass the Appreciated Gateway without being original in the structural sense.

---

## Gate Test

**XI. Appreciation Capture Test**

> Has this relationship been stated qualitatively before, and does this paper supply the first explicit number?

| Question | Answer required to pass |
|----------|------------------------|
| Is the underlying idea already qualitatively accepted? | Yes |
| Does this paper provide the first quantitative calibration, bound, or exclusion? | Yes |
| Is the number reusable by another researcher in a different analysis? | Yes (strengthens classification) |

**If all three: classify as** `formalization of an appreciated idea`
**Not:** repetition. Not: derivation. Not: original structure.

---

## Classification Output

```
Appreciated Gateway: PASS
Classification: Formalization of appreciated idea
Contribution type: First quantitative specification of a qualitatively known relationship
Publishable unit: The number itself, stated as a bound or exclusion
```

## Applied — Hubble Paper (Eric D. Martin)

| Element | Status |
|---------|--------|
| "Ωm prior affects H₀" — qualitatively appreciated? | Yes — broadly held |
| First explicit measured bound in this pipeline? | Yes — ΔH₀ = −0.218 ± 0.005 km/s/Mpc |
| Accounts for only ~3.9% of the gap? | Yes — quantified exclusion |
| Reusable by others? | Yes — slope dH₀/dΩm ≈ −11.6 is portable |

**Appreciated Gateway: PASS**

The centerpiece is not the intuition. It is the measured bound that rules out the prior shift as a primary explanation.

## Keeper Assignment

The Appreciated Gateway must be evaluated by **Eric**, not the LLM.

The LLM cannot determine whether the qualitative idea was "already broadly held" — that requires domain state-space knowledge. The LLM can flag candidate results for Eric to evaluate, but the gate itself requires a human keeper with knowledge of the field's prior discourse.

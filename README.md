# GITGAP

**Research Gap Detection using the Appreciated Gateway**

This repo tracks the methodology and applied tests for finding high-impact, publishable gaps in current literature — specifically "Appreciated Theories": concepts a community broadly accepts but has not yet locked down with a portable, reusable number.
